**Дано:** (y'''-4y'=\sin x)

### 1) Характеристическое уравнение (для однородного)

Однородное: (y'''-4y'=0).

Характеристический многочлен:
[
r^3-4r=r(r^2-4)=r(r-2)(r+2)=0
]
Корни: (r_1=0,; r_2=2,; r_3=-2). ([Википедия][1])

### 2) Общее решение однородного уравнения

[
y_h=C_1+C_2e^{2x}+C_3e^{-2x}.
]

### 3) Частное решение (метод неопределённых коэффициентов)

Так как правая часть (\sin x), пробуем:
[
y_p=A\sin x+B\cos x.
]
Тогда
[
y_p' = A\cos x - B\sin x,\qquad
y_p''' = -A\cos x + B\sin x.
]
Подставляем в (y'''-4y'=\sin x):
[
(-A\cos x + B\sin x)-4(A\cos x - B\sin x)=(-5A)\cos x + (5B)\sin x=\sin x.
]
Сравниваем коэффициенты:
[
-5A=0 \Rightarrow A=0,\qquad 5B=1 \Rightarrow B=\frac15.
]
Значит,
[
y_p=\frac15\cos x.
]
(Идея метода: угадываем вид (y_p) по правой части и подбираем коэффициенты.) ([Pauls Online Math Notes][2])

### 4) Общее решение ЛНДУ

[
\boxed{,y(x)=C_1+C_2e^{2x}+C_3e^{-2x}+\frac15\cos x, }.
]
